The JSON format is quite intuitive. The `theme` and `type` fields are optional. The app has the following themes available

* `"theme": "golden-hour",`
* `"theme": "ocean-deep",`
* `"theme": "purple-dream",`
* `"theme": "arctic",`
* `"theme": "cotton-candy",`
* `"theme": "lavender",`
* `"theme": "sunset",`
* `"theme": "rose-gold",`
* `"theme": "skyblue",`
